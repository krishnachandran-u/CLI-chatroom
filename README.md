# CLI-Chatroom

## Usage

Clone the 'CLI-chatroom' repository using git:

```bash
git clone https://github.com/krishnachandran-u/CLI-chatroom.git 
```

### Start server

Run the server on a single machine:

```bash
npm run server
```

### Start client

Run client:

```bash
npm run client
```

## Commands

Send this to server to close the client:

```bash
!logout
```
```
